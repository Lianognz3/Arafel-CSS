Arafel CSS

Un micro-framework ligero, elegante y moderno para tus proyectos web.


Descargar Arafel CSS

Instalación

Descarga el archivo .zip, extráelo y enlaza el archivo arafel.min.css en tu archivo HTML:

<link rel="stylesheet" href="dist/arafel.min.css" />

Botones

Agrega la clase .button y una clase de color para obtener botones modernos con efectos de elevación y transiciones suaves.

Botones con fondo (Sólidos)

<button class="button azul-con-fondo">Azul</button>

Botones sin fondo (Huecos)

<button class="button rojo-sin-fondo">Rojo</button>

Campos de Texto (Inputs)

Usa la clase .input para aplicar estilos modernos a tus formularios. Incluyen un elegante efecto de enfoque (focus).

<input type="text" class="input" placeholder="Tu nombre...">

Loaders / Indicadores de Carga

El framework incluye múltiples indicadores de carga construidos puramente en CSS, ahora con nombres en español.

<!-- Anillo doble -->
<div class="lds-anillo-doble"></div>

<!-- Corazón -->
<div class="lds-corazon"><div></div></div>

<!-- Onda (Ripple) -->
<div class="lds-onda"><div></div><div></div></div>

Sistema de Cuadrícula (Grid Responsivo)

Usa .ancho-* para definir el porcentaje de ancho (desde 1 hasta 100).

Además, Arafel soporta diseño responsivo con modificadores: .ancho-sm-* (pantallas > 576px), .ancho-md-* (> 768px) y .ancho-lg-* (> 992px).

.ancho-100 .ancho-md-50
.ancho-100 .ancho-md-80 

<!-- En móviles ocupa el 100%, pero en pantallas medianas (md) ocupa el 50% -->
<div class="ancho-100 ancho-md-50 azul-con-fondo">Responsivo</div>
